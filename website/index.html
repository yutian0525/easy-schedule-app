导入AI IMPORT

不用一节一节手填

把课表截图交给 AI，三步搬进 App，导入时自动检测时间冲突。

1

复制提示词

在「课表编辑」里点 复制提示词，一段为本 App 写好的指令就在剪贴板里了。

2

连同截图发给 AI

把教务系统的课表截图和提示词一起发给任意大模型，它会吐出规范的 JSON。

3

粘贴 JSON 导入

点 粘贴 JSON 导入，课程批量入库，冲突的节次会被自动标出。
